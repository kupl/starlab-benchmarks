O que é?

O Caelum Stella visa suprir as necessidades comumente encontradas em aplicações desenvolvidas para o mercado brasileiro. Atualmente, o Caelum Stella fornece uma biblioteca de validadores, formatadores e conversores para documentos brasileiros, tais como CPF, CNPJ e PIS/PASEP; além de um gerador de boletos bancários.

O Stella também fornece adaptadores para JSF, JSP tag-libs e Hibernate Validator.

Validadores

Validação é uma ferramenta poderosa. Ela permite verificar rapidamente se um valor está de acordo com o padrão esperado e rejeita prontamente os que não se encaixam na regra. Dessa forma é possível ter uma garantia sobre a integridade dos dados, ou seja, determinar se os dados recebidos são os mesmos que os enviados. Se houver algum problema com os dados, saiba o quanto antes.

Atenção: Os validadores fornecidos validam a integridade da combinação dos dígitos fornecidos para determinado documento, e não o documento em si.

Formatadores

Os formatadores são reponsáveis pela transformação das possíveis representações de um documento, por exemplo: um CPF representado na forma XXX.XXX.XXX-XX pode ser transformado para a representação sem pontos e traços, e vice-versa.

Gerador de Boletos Bancários

O gerador de boletos bancários é uma ferramenta que permite a rápida criação de boletos bancários, de maneira simples e intuitiva.
