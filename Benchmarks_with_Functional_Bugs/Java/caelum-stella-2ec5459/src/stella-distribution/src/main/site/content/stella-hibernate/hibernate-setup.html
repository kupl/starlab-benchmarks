Configuração do Ambiente

Para usar o Stella Hibernate é necessário fazer o download do Hibernate Core, Hibernate Annotations e Hibernate Validator, disponíveis em http://www.hibernate.org/6.html

Lista de jars necessários no classpath

  • caelum-stella-core-1.x.jar
  • caelum-stella-hibernate-validator-1.x.jar

  • commons-collections-2.1.x.jar
  • hibernate-3.2.x.jar
  • hibernate-annotations-3.3.x.jar
  • hibernate-commons-annotations-3.3.x.jar
  • hibernate-validator-3.0.x.jar
  • ejb3-persistence.jar (JPA 1.0)

  • jta-1.0.1B.jar

Arquivos de configuração

Para configurar as mensagens de erro utilize o arquivo ValidatorMessages.properties. Para saber mais sobre como configurar este arquivo, leia o guia de referência do Hibernate Validator.
