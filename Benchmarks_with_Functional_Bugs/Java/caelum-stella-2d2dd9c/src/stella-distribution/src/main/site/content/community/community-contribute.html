Introdução

O Caelum Stella é um projeto de código aberto, e as contribuições da comunidade são sempre bem vindas. Os usuários podem baixar a biblioteca e começar a contribuir prontamente. Além do mais, podem também enviar dúvidas e sugestões para a lista de discussão do projeto, no fórum do GUJ ou no JIRA do projeto.

O número de colaboradores está aumentando e queremos motivá-los e orientá-los para que o projeto continue crescendo.

Primeiros Passos

O primeiro passo para contribuir com Stella é se cadastrar na lista de desnovolvedores. Tendo isto feito, chegou a hora de instalar o ambiente de desenvolvimento. Para isto, é necessário instalar na sua máquina o Subversion (para controle de versão), o Maven2 (para gerenciamento das dependências e automatização de tarefas) e o Eclipse. Além disso, é conveniente instalar um plugin no eclipse chamado Subversive.

Caso tenha dificuldade para instalar estas ferramentas, envie um e-mail para a lista de desenvolvedores. Para alguns, este início pode ser um pouco complicado, mas esta é uma oportunidade para aprender.

Para participar do Caelum Stella não é necessário ler toda a documentação inicialmente. No entanto, é recomendável que se leia documentação referente ao Stella Core e ao Stella Boleto, para entender como a API funciona.

Baixando o código do repositório

Com o cliente do subversion instalado, execute a linha de comando abaixo para baixar o código do Caelum Stella para a sua máquina. Execute o comando na pasta onde deseja salvar os arquivos. Uma nova pasta chamada caelum-stella será criada.

svn co http://caelum-stella.svn.sourceforge.net/svnroot/caelum-stella/trunk caelum-stella

Gerando os arquivos do eclipse

Garanta que o maven está instalado e disponível no PATH. Para isso, execute:

mvn --version

A saída deve ser parecida com esta:
Maven version: 2.0.9
Java version: 1.5.0_16
OS name: "mac os x" version: "10.5.6" arch: "i386" Family: "unix"

Com o maven instalado, podemos agora utilizá-lo para montar os arquivos de projeto do eclipse. Para isso, na pasta caelum-stella, execute a seguinte linha de comando:

mvn eclipse:eclipse

Aguarde. Aguardade até que o maven baixe todas as dependências dos projetos do e salve-as na pasta ~/.m2/repository . Isto deve levar alguns minutos.

Após a execução do comando, devem ter sido criados arquivos (.classpath e .project) nas pastas dos projetos.

Agora precisamos, importar os projetos criados para o Eclipse.

Importando os projetos no Eclipse

Abra o Eclipse e clique em File -> Import ... . Selecione a opção "Existing projects into workspace" e clique em Next. Na opção "Select root folder:" insira o caminho correto para a pasta "caelum-stella". Selecione todos os projetos disponíves e pressione o botão "Finish".

O processo está quase pronto. Por último precisamos configurar um variável no Eclipse que indica onde está o repositório local do Maven, isto é, aquela pasta ~/.m2/repository onde as dependências foram salvas. Para isto, selecione no menu a opção Window -> Preferences . Na aba da esquerda, selecione o item Java --> Build Path --> Classpath Variables. Clique no botão "New ..." que aparece na aba da direta. Na janela que se abriu, atribua o valor M2_REPO no campo Name, e algo como /Users/leonardobessa/.m2/repository no campo Path.

O Eclipse irá perguntar se deseja fazer o build do projeto novamente. Diga que sim.

Pronto. O ambiente de desenovlimento do Caelum Stella está montado.
