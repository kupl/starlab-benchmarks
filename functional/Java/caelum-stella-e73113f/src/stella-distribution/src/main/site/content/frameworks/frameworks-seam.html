Validando na camada de apresentação

Utilizando o JSF com o JBoss Seam™, você pode ativar o processo de validação na camada de apresentação usando as tags JSF do Seam: <s:validate> e <s:validateAll>. Dessa forma é possível expressar as regras de validação no domínio, e as violações serão expressas na camada de apresentação.

pagina.xhtml

Além disso, ao utilizando Ajax4JSF™ a validação poderá ser feita no lado do cliente com a adição de apenas algumas Tags JSF.

Para maiores informações acesse a documentação do JBoss Seam.
