Funcionalidades Planejadas (Roadmap)

Esta lista contém as funcionalidades desejadas para as próximas versões do Caelum Stella. Caso você tenha alguma sugestão, não deixe de opinar na nossa lista de discussão, GUJ, ou diretamente com os desenvolvedores.

  • Validador: Reservista Exército

  • Validador: CEP
  • Lista com nomes e códigos dos bancos
  • Lista de Países, Cidades e Estados com DDI e DDD

Stella Web-Services

  • Rotinas para facilitar assinatura digital eCPF
